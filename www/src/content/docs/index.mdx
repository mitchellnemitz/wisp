## Shell scripting without the footguns

The compiler inserts every quote correctly and enforces an explicit type system, so the quoting, word-splitting, and implicit-coercion bugs that plague hand-written shell become structurally impossible.

```wisp
fn main() -> int {
    print("hello, world")
    return 0
}
```
